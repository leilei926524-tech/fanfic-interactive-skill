# tongren-wenyou

一个可在 Claude Code 里直接调用的本地自定义 skill，用来运行中文同人文互动剧情游戏。

## 调用方式

在当前工作区中输入：

```text
/tongren-wenyou
```

也可以直接带开局设定，例如：

```text
/tongren-wenyou 我要玩一个仿红楼梦的文游，我是寄居贾府的表姑娘，主攻略对象偏清冷克制。
```

## 适用玩法

- 原创古风乙女文游
- 指定同人 IP 文游
- 群像关系拉扯
- 多分支多结局互动叙事
- 可查状态、存档摘要、当前目标

## 内置能力

- 主持剧情推进
- 维护隐藏状态
- 角色好感/信任/怀疑/执念变化
- 章节推进和锁线
- 多结局判定

## 游戏内可直接输入

- `状态`
- `详细数值面板`
- `存档摘要`
- `当前目标`

## 后续修改建议

如果你想把它进一步改成某个固定 IP 专用 skill，可以重点改这几个部分：

1. `SKILL.md` 里的“默认世界观模板”
2. `SKILL.md` 里的“角色类型默认包含”
3. `SKILL.md` 里的“结局系统”
4. `SKILL.md` 里的“开场模板”

## 参考思路

这个 skill 的结构参考了你给的两个项目常见做法：

- 主持提示词集中在一个主文件
- 使用隐藏状态系统驱动剧情
- 用章节/线路/结局思路来控制分支
- 让用户可以直接开玩，也可以快速定制
