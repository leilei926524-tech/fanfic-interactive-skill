# 同人文游skill

把 Claude Code 变成你的专属文游主持人。

这是一个面向中文用户的 Claude Code 自定义 Skill。安装后，你可以直接在对话里开启一场可互动、可分线、可查状态、可打出多结局的同人文剧情游戏。

它不是一段一次性 prompt，而是一套为“可玩性”设计过的文游主持规则：能接住自然语言行动，能维护隐藏状态，能推进人物关系，也能在甜、虐、误会、真相和抉择之间稳定运转。

## 它适合谁

- 想把 Claude Code 直接当文游引擎来玩的用户
- 想做乙游 / 同人 / 古风 / 群像互动剧情的人
- 想把一个灵感快速变成可试玩文本游戏的人
- 想在现有 IP 上跑角色互动、感情线和多结局的人

## 这个 Skill 能带来什么体验

### 1. 真正可玩的互动感
你不需要按照固定按钮点选。

你可以直接输入：
- “我不接他的话，只看他一眼。”
- “我假装答应，回去再查那封信。”
- “我在宴席上故意提起她的名字试探他的反应。”
- “我现在就跟他摊牌。”

Skill 会把这些自然语言动作当成有效操作，继续推进剧情，而不是把你拉回机械选项。

### 2. 有分支，不是空转陪聊
它内部维护章节、角色关系、危险值、真相进度、流言压力和关键事件标记。

这意味着：
- 你的靠近和回避都会留下后果
- 不同角色会形成不同线路
- 情感推进和真相推进会彼此影响
- 失败、错过、误判也能成为剧情的一部分
- 最终可以通向真结局、个人线结局、遗憾结局、坏结局或隐藏结局

### 3. 既能原创，也能跑同人 IP
默认是古风情感向世界观，但你可以在开局时直接指定：
- 某个作品 IP
- 某个角色
- 某种关系线
- 甜虐风格
- 穿越 / 重生 / 黑化 / 反转等玩法

它会优先服从你的设定，不把你强行拉回默认模板。

## 如何使用

把这个项目放进 Claude Code 工作区后，直接调用：

```text
/tongren-wenyou
```

也可以带设定直接开始：

```text
/tongren-wenyou 我要玩一个仿红楼梦的同人文游，我是寄居贾府的表姑娘，主攻略对象偏清冷克制，整体偏虐。
```

## 游戏内可直接输入的指令

- `状态`
- `详细数值面板`
- `存档摘要`
- `当前目标`

## 开局方式

Skill 支持 3 种开局：

1. 直接开玩
   - 使用默认古风情感线，立即进入第一幕。
2. 快速定制
   - 指定题材、身份、攻略对象和甜虐偏好。
3. 指定同人 IP
   - 直接说作品名、角色名和想玩的关系线。

## 示例

### 示例 1：默认古风开局

```text
/tongren-wenyou
```

### 示例 2：指定风格

```text
/tongren-wenyou 我想玩偏慢热、偏压抑、后期爆发的古风线，男主是嘴硬心软型。
```

### 示例 3：指定同人 IP

```text
/tongren-wenyou 我想玩一个宝黛钗相关的互动文游，我是新来的表姑娘，整体走群像拉扯。
```

## 适合二次创作的方向

你可以很容易把它改造成：

- 固定 IP 专用文游 Skill
- 固定男主专线 Skill
- 民国 / 西幻 / 学院 / 宫廷版本
- 更甜版 / 更虐版 / 更慢热版 / 更高危版
- 侧重悬疑真相或侧重感情拉扯的版本

## 为什么这个项目值得收藏

因为它解决的不是“怎么写一段文游 prompt”，而是“怎么让一个文游真的能连续玩下去”。

它更重视：
- 玩家输入的承接能力
- 人设稳定性
- 状态变化的合理性
- 分支推进的连续感
- 结局条件的可积累性

换句话说，它更像一个能长期跑剧情的互动框架，而不是只适合玩 3 轮就散掉的灵感片段。
